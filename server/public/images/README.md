# Electronics Shop Images

Store your product images here.

## How to Use:

1. **Add images** to this folder:
   - Place any `.jpg`, `.png`, `.webp` image files here
   - Example: `phone.jpg`, `laptop.png`, `earbuds.webp`

2. **Use in Admin Panel**:
   - Image URL format: `http://localhost:5000/images/filename.jpg`
   - Example: `http://localhost:5000/images/samsung-s24.jpg`

3. **Image Recommendations**:
   - Size: 300x300px to 500x500px (square is best)
   - Format: JPG or PNG
   - File size: Under 500KB (for fast loading)

## Examples:

- `http://localhost:5000/images/phone.jpg`
- `http://localhost:5000/images/laptop.png`
- `http://localhost:5000/images/earbuds.webp`
